## Fitur & Spesifikasi Proyek
Berikut adalah spesifikasi tugas yang telah diimplementasikan dalam aplikasi ini:

* **Sistem Otentikasi:** Fitur Login & Logout Sistem.
* **Manajemen Data (CRUD) & Upload:** Proses Tambah, Edit, Hapus data yang dilengkapi dengan fitur *Upload Multiple File*.
* **Pencarian & Datatable:** Fitur Pencarian Data untuk mempermudah pengelolaan data di dalam tabel.
* **Tanda Tangan Digital:** Implementasi *Canvas* untuk fitur tanda tangan (TTD) digital.
* **Multimedia:** Integrasi Video / Animasi dan komponen Audio di dalam aplikasi.
* **Komponen UI:** Penggunaan *Modal* untuk interaksi user interface yang lebih interaktif.
* **Database:** Koneksi menggunakan database MySQL (`db_labborrow.sql`).
